#### Restrictions on Primary Data Sharing

The primary operational dataset underlying this review comprises **private internal reservation, fleet, and service-log records** from a rent-a-car operator and is not publicly available due to commercial confidentiality and data-protection obligations. This restriction applies to the raw input data only. All derived extraction data (aggregated at the study level), analysis code, and the PRISMA flow diagram remain openly available as described above.

#### FAIR Data Principles

Deposited materials comply with the FAIR data principles ([TRAINING-DATA CLAIM]):

- **Findable:** All deposited files are assigned a persistent identifier (DOI) and described by structured metadata within the repository record.

- **Interoperable:** Extraction data are provided in CSV format with column headers conforming to standard variable naming; bibliographic records are available in RIS format compatible with reference management software.
- **Reusable:** Each deposited file is accompanied by a data dictionary defining all variable names, coding schemes, and units; the data license (see below) explicitly permits reuse, modification, and redistribution with attribution.
